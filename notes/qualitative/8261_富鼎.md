## 業務概況

富鼎(富鼎先進電子)為MOSFET IC設計公司,產品涵蓋低壓/中壓/高壓MOSFET、IGBT模組、電源管理
IC(Dr. MOS)及風扇驅動IC。**商業模式為銷售自有設計IC產品**(非代工服務費)。2022年國創半導體
入股取得約三成股權、躍居第一大股東,此舉被視為富鼎切入車用市場的重要契機。

## 客戶/產品結構(2025年Q3)

| 分類 | 占比 | 備註 |
|---|---|---|
| 低壓MOSFET | 50% | |
| 中壓MOSFET | 32% | |
| 高壓MOSFET | 16% | |
| IGBT | 2% | 含1,200V高壓IGBT模組 |

應用別:風扇散熱36%(季度年增287%,已成為最大成長引擎與毛利貢獻主力)、電源供應器(SPS)35%
(因競爭加劇年減)、電腦運算18%(受中國PC市場疲軟影響下滑)。主要客戶以台系電源供應器與散熱
大廠為主,並透過與國創半導體的策略合作拓展車用市場。

## 財務亮點(質化色彩,非計分依據)

- 2025年Q3合併營收8.3億元(季增1.5%、年增3%),毛利率36.8%(季降1.1個百分點);稅後淨利
  2.28億元(季增188%),EPS 1.93元(年增47%)。
- 2025年前三季累計營收26.64億元,毛利率37.2%(年增10.5個百分點)。
- 2026年Q1 EPS 1.57元;2026年5月單月營收2.97億元(月增4.55%、年增1.15%)。
- 營收曾於2023年大幅下滑(-27%),2024年強力反彈(+34%),顯示已走出谷底、恢復成長動能。

## 成長動能與技術亮點

- 風扇散熱應用單季營收年增287%,達3.1億元(2025Q3),為近期最大成長引擎。
- 1,200V高壓IGBT模組已量產並通過客戶驗證,產品線持續向高壓/高毛利延伸。
- 電源供應器(SPS)市場為2026年主要成長策略,搭配PC及周邊應用新產品於台灣、中國市場發酵。
- 高整合型Dr. MOS(Driver + MOSFETs)新產品專攻AI伺服器電源應用。

## 風險與觀察重點

- Q4進入傳統淡季,客戶拉貨動能趨緩,營收具明顯季節性,單季數字需搭配季節調整解讀。
- 高壓產品線面臨歐美大廠激烈競爭;中低壓MOSFET市場競爭白熱化,法說會形容為「殺戮市場」,
  價格壓力大,可能侵蝕毛利率。
- 電腦運算應用受中國電競PC市場需求疲軟影響下滑,該應用別占比正被風扇散熱/SPS取代,產品
  組合調整過程中營收波動可能加大,不宜單季判斷趨勢。

## 法說會頻率

歷年多為1年1場(2017~2024年間除2022年2場外均為1場),2025年驟增至6場,2026年至今2場,
推測因AI伺服器電源題材升溫、法人關注度提高而加密場次(如2025-10-30場次即對應Q3財報)。
公司規模仍屬中小型,建議至少每季關注一次,以Q3(10月底,歷史上已確認有場次)與年報公布後
(3~4月)作為核心更新節點。
