## 30 秒摘要

- M31 是無自有 IC 產品的純矽智財供應商，主力為 Foundation IP（Standard Cell、I/O、Memory Compiler）、高速介面 IP 及 IP 整合服務；客戶橫跨晶圓代工廠與 Fabless，收入來自滿足履約義務後認列的技術服務／授權金，以及客戶晶片量產後的權利金。[S1][S4]
- 2026Q1 營收 3.947 億元、年減 9.2%，營業損失 6,627 萬元、稅後淨損 4,925 萬元、EPS -1.18 元；審計口徑的技術服務收入占 81.9%、權利金占 18.1%，尚未達管理層全年「站上 20%」的目標。[S1][S2][S3]
- 最大成長選擇權是 2–8nm 專案從授權轉為量產權利金：公司稱既有 2nm 客戶已 tape-out，目標在 2026 年底或 2027 年初看到權利金推升；但這是前瞻敘事，量產時點、良率、晶圓投片與權利金絕對金額才是驗證點。[S2][S3]
- 結構性優勢是矽驗證 IP、晶圓廠平台合作與橫跨 2–180nm 的節點經驗；結構性弱點是每個新節點約需兩年設計驗證、品牌與產品線規模小於既有競爭者，且 2025 年研發費用約占營收 75.8%，營運槓桿對授權專案時點很敏感。[S4]
- 當前首要觀察為 Q2 營收與 EPS 能否證實遞延案回補、權利金占比與絕對額能否同步上升、逾期應收的備抵是否繼續增加，以及股東會授權最高 1,600 萬股長期籌資額度會否實際執行。[S1][S3][S5]

## 產業位置與競爭格局

M31 位於半導體設計生態系的「可重複使用、已經製程驗證的 IP block」層：對晶圓廠提供特定製程的 Foundation IP 平台，對 IC 設計公司提供標準或客製 Foundation IP、Interface IP 與整合服務。公司自述不發展自有 IC、不與客戶競爭，因此 Universe 歸屬應是矽智財而不是 ASIC 設計服務或晶片產品公司。[S4]

Foundation IP 與製程工藝緊密綁定；Interface IP 則含複雜類比電路，也必須在各製程節點上驗證。公司估計一個節點世代約需兩年；這使「晶圓廠平台入選＋矽驗證＋客戶已完成整合」形成轉換成本，但也代表新節點投入到權利金貢獻有長延遲。[S4]

競爭壓力來自既有全球 IP 商與中國本土 IP 供應鏈。M31 在年報承認成立較晚、營運規模較小、產品線較少與品牌知名度較低；其因應是延伸產品線、發展差異化 IP 與整合服務。年報依第三方 IPnest 數據自估 2025 全球市占約 0.58%，應視為公司引用外部資料後的估算，而非本次研究另行驗證的市占率。[S4]

## 業務與獲利模式

- **技術服務／授權金：**客戶合約由公司在滿足履約義務時認列對價；晶圓廠型 Foundation IP 可能在新製程開發期就開始認列授權收入，專案導向使單月與單季收入受簽約、驗收時點影響。[S1][S3][S4]
- **權利金：**標準化 IP 授權給客戶後，只在客戶使用該 IP 的晶片量產後，依合約收取權利金。因此權利金是早期研發與授權案真正轉為可重複收入的滯後指標，不能用 tape-out 或單一「已採用」公告直接當成量產業績。[S1][S4]
- **100% 毛利率的解讀：**財報未單列營業成本，所以營收全數列為毛利；但核心產品成本以研發費用列在營業費用。2025 年研發費 13.506 億元，約為營收 75.8%；2026Q1 研發費 3.329 億元，約為營收 84.3%。因此 100% 會計毛利率不等於高營業利率。[S1][S4]
- **經營槓桿：**研發、EDA 工具與人才費用需先行投入，短期獲利取決於授權金專案是否按時驗收；中長期獲利品質則取決於已授權 IP 進入量產的權利金量與製程組合。[S1][S3][S4]

## 財務品質與資本配置

2025 年營收 17.822 億元、年增 20.3%，營業利益 0.932 億元；稅後淨利 0.706 億元、年減 44.4%，公司歸因於台幣升值造成兌換損失。全年營業現金流入只有 0.263 億元，約為淨利的 37%，顯示即使年營收創高，現金轉換仍不強。[S4]

2026Q1 營收 3.947 億元、年減 9.2%；營業費用 4.610 億元、年增 8.0%，其中研發費 3.329 億元。營業損失 0.663 億元、淨損 0.492 億元。公司將營收不如預期歸因於美國 Fabless 專案遞延與農曆工作天數較少；這項解釋必須用 Q2 實際回補驗證，不能先當成已恢復。[S1][S2]

Q1 營業現金淨流出 1.371 億元，主要營運資金變動包含預付款增加 0.635 億元、其他應付款減少 0.707 億元。期末現金 3.019 億元；現金加流動金融資產為 4.431 億元，總負債 2.232 億元、權益 18.020 億元。資產負債表未列計息借款，負債主要由應付股利、應付帳款／其他應付款、所得稅與租賃負債構成；短期償債壓力不高，但若營業現金持續流出，仍會消耗這個緩衝。[S1][S2]

應收帳款淨額由 2025 年底 3.356 億元降至 Q1 的 2.810 億元，但毛額 3.173 億元中已提備抵 0.362 億元（約 11.4%）；逾期超過 365 天的 0.285 億元已全數提列，Q1 預期信用減損 869 萬元，高於上年同期 68 萬元。這是比「帳上應收下降」更重要的品質警訊。[S1]

2025 年度盈餘配發現金股利每股 1.2 元、總額 5,015 萬元。2026-05-22 股東會另通過長期籌資授權：一年內可在普通股不超過 1,600 萬股額度內，以公募現增、私募普通股或私募可轉債擇一或搭配。以 2026-03-24 流通在外 4,179.8 萬股計，額度上限約等於現有股數 38.3%；這不代表會全數發行，但是需持續追蹤的潛在稀釋與資本配置事件。[S4][S5]

## 優勢、弱點與護城河

- **製程共優化與矽驗證經驗：**Interface IP 授權節點從 2nm 到 180nm，權利金節點從 3nm 到 180nm；Foundation IP 需與晶圓廠工藝共同開發。這些已驗證資產可降低客戶整合與 tape-out 風險，但護城河只有在持續跟上新製程且客戶最終量產時才能變現。[S4]
- **兩類 IP 與整合能力：**Foundation IP 累積製程平台深度，Interface IP 與 Controller 整合則增加單案內容與客戶切換成本。公司 2025 年完成多項 N3／N3P、N4P、N5、N6 等記憶體、PCIe、M-PHY、ONFI 技術，但「開發完成」不等於商業量產。[S4]
- **權利金的可擴張性：**已開發 IP 隨客戶量產可持續收取權利金，且不需承擔晶圓製造資本支出；年報也確認公司沒有實體商品生產廠房。反面是量產決策、投片量與終端需求不由 M31 控制。[S1][S4]
- **規模與品牌弱點：**公司自述成立較晚、規模較小、產品線較少、品牌知名度較低；同時中國本土 IP 供應鏈可能加大區域競爭與價格壓力。[S4]
- **高固定投入與長回收期：**2025 年研發費 13.506 億元，研發人數 331 人；至 2026-02-28 增至 339 人。單一授權專案延遲就可使單季轉損，顯示在權利金規模化以前，研發槓桿同時是優勢與風險。[S1][S4]

## 成長動能、催化劑與驗證 KPI

- **2nm 客戶量產：**管理層稱已有 2nm 客戶完成 tape-out，並將 2026 年底或 2027 年初視為權利金可能推升的時間窗。**KPI：**2–8nm 權利金絕對額、量產客戶數、季度權利金占比；**失效條件：**2027Q1 後仍只有授權或 tape-out 敘事，沒有對應權利金改善。[S2][S3]
- **權利金結構升級：**Q1 權利金占 18.1%；公司年度目標是站上 20%，並稱小於 16nm 的權利金年增近 700%、22nm 以上成熟製程年減約 20%；公司未揭露這兩個製程區間的權利金絕對額。**KPI：**權利金絕對額年增、占比能否連續兩季高於 20%；**失效條件：**只靠授權金縮減使占比上升，權利金絕對額未增長。[S2][S3]
- **Q1 遞延案回補：**管理層目標是 Q2 EPS 轉正，並維持 2026 全年營收雙位數成長目標。**KPI：**Q2 營收是否高於 Q1、美國地區占比是否回升、Q2 營業利益與 EPS；**失效條件：**Q2 仍處營業損失或公司再次將案件推延至下季。[S2][S3]
- **介面 IP 技術里程碑：**公司宣布台積電 N2P eUSB2V2 已 tape-out，4nm M-PHY v5.0 已 silicon-proven、單 lane 23.32Gbps，並可與 UFSHCI v4.1、UniPro 組成 UFS 4.1 方案。**KPI：**第三方客戶 design win、新授權數、量產與權利金；**失效條件：**長期只有技術完成公告，沒有客戶採用或收入轉換。[S2]
- **費用紀律：**管理層稱前期 EDA 大額投入已告一段落，2026 年營業費用增幅目標控制在 10% 以內或附近；Q1 實際年增 8.0%。**KPI：**每季營業費用年增、研發費率、每人營收；**失效條件：**營收尚未回升而費用增幅連續超過 10%。[S1][S3]

## 風險與預警指標

- **專案時點與單季波動：**公司明確稱專案導向使單月數字容易被誤讀；Q1 已顯示美國案件遞延可將公司推入損失。**預警：**連續兩季營收年減、專案再遞延、合約負債或新授權數不增。[S1][S3]
- **權利金轉換不如預期：**財報收入政策與管理層說明都顯示，授權或 tape-out 並不等於已產生權利金，仍須客戶進入量產；因此研究上不能把技術里程碑直接當成業績。**預警：**先進製程授權占比高，但權利金絕對額、現金流與獲利不改善。[S1][S2][S3]
- **應收帳款與信用損失：**Q1 備抵達毛應收約 11.4%，且當季減損費用顯著上升。**預警：**逾期 181 天以上款項、備抵率、減損費用與回收現金。[S1]
- **地區與出口管制：**Q1 中國收入占 46.4%；公司承認美中科技與貿易政策、出口管制、關稅與投資審查是重要變數。**預警：**中國案件取消、應收回收拉長、公司客戶審查或管制政策更新。[S2][S4]
- **匯率：**Q1 美元貨幣性資產淨暴險大，財報估美元相對台幣變動 1% 對稅前損益影響約 657 萬元；2025 年獲利也已受台幣升值壓縮。**預警：**台幣快速升值、兌換損益占稅前利益過高。[S1][S4]
- **人才與 EDA 投入：**先進製程需稀缺工程人才、高額設計工具與長期驗證，若費用先增而量產延遲，獲利將再受壓力。**預警：**研發費率、離職與擴編速度、每季營業費用年增。[S3][S4]
- **潛在稀釋：**股東會已授權最高 1,600 萬股等值額度的現增／私募／可轉債籌資。**預警：**董事會啟動籌資、發行折價、實際新股數、募集資金用途與報酬。[S5]

## 管理層與治理

董事長陳慧玲在 2026-03-24 持有 878.7 萬股、持股 21.02%，是最大股東；總經理張原熏負責營運。2026-05-22 股東會改選第六屆董事會，共4 名董事與 3 名獨立董事，任期至 2029-05-21。股權集中有利於長期研發決策連續性，但最大股東同時任董事長，仍需靠獨立董事、審計委員會與資本配置透明度制衡。[S4][S5]

2025 年股東會通過的長期資金募集案，董事會於 2026-03-03 決議不繼續辦理；但 2026 股東會又通過新的 1,600 萬股上限額度，贊成 86.67%、反對 12.07%。近期治理重點不是「有無授權」，而是董事會若執行，是否揭露明確投資項目、價格、對象、預期報酬與對既有股東稀釋。[S5]

公司稱已連續四年位於櫃買中心公司治理評鑑前 5%，並自 2019 年第四季起於董事會後公告中英文財務報表。這些是正向流程證據，但不取代對籌資、股份基礎給付、應收減損與管理層預測達成率的個別審查。[S4]
