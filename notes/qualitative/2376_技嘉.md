## 30 秒摘要

- 技嘉同時經營主機板、顯示卡、筆電等消費型硬體，以及伺服器、機櫃與資料中心解決方案；2026Q1 的 SERVER 產品線已占合併營收 71.5%，高於 2025Q1 的 55%，因此 Universe 依「一檔一群」歸入 serverodm。[S1][S3]

- 71.5%是整體 SERVER 產品線，不是 AI server 占比；同一份法說另稱 2026Q1 主要成長動能來自 AI server、該需求年增 110%，但未揭露 AI server 的獨立營收金額或占比。[S3]

- 2026Q1 營收 1,050.57 億元、年增 59.78%，營業利益 74.13 億元、年增 74.87%，EPS 7.86 元；營收與獲利創高，但毛利率 12.01%低於去年同期 12.90%，顯示高速成長並非所有利潤率同步改善。[S2][S3]

- 最大動能是 GB300 NVL72、液冷 GPU server 與 GIGAPOD 整合方案；主要弱點則是存貨年增 124.88%、關鍵晶片供應集中，以及 AI-only 收入不可見。核心追蹤指標應是 SERVER 占比、毛利率、存貨／應付帳款、營業現金流與新平台量產節奏。[S1][S2][S3]

## 產業位置與競爭格局

技嘉位於運算硬體設計、製造與品牌／通路端，伺服器產品涵蓋 AI、HPC、Cloud、5G，並延伸至機櫃、液冷、浸沒式冷卻及整合式資料中心；這使其經濟曝險不只在單機 ODM，也包含平台設計、系統整合與部署。[S1][S3]

年報把研發設計、零組件採購、SMT／DIP、測試、品質檢驗與出貨列為主要產製流程，顯示公司以硬體產品銷售為主，不是收取純工程服務費；法說的 GIGAPOD 與管理平台則代表價值鏈正由單機向叢集與軟硬整合延伸。[S1][S3]

公司自述伺服器市場受 AI、雲端與 5G 運算需求推動，但核心文件沒有同業市占、客戶轉換成本、認證週期或可比報價；本稿因此只驗證產品覆蓋與收入結構，不把「領先」或獎項直接視為已量化的護城河。[S1][S3]

## 業務與獲利模式

收入來自 SERVER、顯示卡、主機板及其他運算硬體銷售；年報列示的伺服器用途是資料中心運算，產品從單機延伸到 OCP 機櫃、液冷／浸沒式冷卻及 GIGA POD，主要獲利仍受出貨量、產品組合、零組件成本與匯率共同影響。[S1][S3]

2026Q1 SERVER 占比 71.5%已超過 R1 的主營收門檻，主機板與顯示卡合計 25.5%；故 serverodm 歸屬是依合併產品結構，而不是因公司同時推出 AI 題材產品。[S3]

2026Q1 毛利率年減 0.89 個百分點至 12.01%，但費用率由 6.45%降至 4.95%，使營業利益率由 6.45%升至 7.06%；當季獲利改善較多來自營收規模與費用率，而不能只歸因於 AI server 毛利。[S2][S3]

## 財務品質與資本配置

- 2026Q1 營收 1,050.57 億元、毛利 126.14 億元、營業利益 74.13 億元、淨利 56.44 億元；營業利益增速高於營收，但毛利增速 48.75%低於營收 59.78%，需持續檢驗產品組合與採購成本。[S2][S3]

- 2026Q1 末現金及約當現金 633.11 億元、年增 58.50%，應收帳款 239.33 億元、年減 22.06%；現金增加與應收下降有利流動性，但仍須以正式現金流量表確認營業現金流與融資／預收因素。[S2][S3]

- 同期存貨 1,316.25 億元、年增 124.88%，應付帳款 665.07 億元、年增 126.05%，兩者增速均明顯高於營收；這與伺服器放量及備料相容，但若後續 SERVER 出貨或平台轉換放慢，庫存與採購承諾會先成為壓力。[S2][S3]

- 2026Q1 末負債 1,807.81 億元、權益 645.67 億元，負債成長 97.73%高於權益成長 13.35%；規模擴張需同步追蹤流動負債、供應商融資、現金流與存貨週轉，而不能只看帳上現金。[S2][S3]

## 優勢、弱點與護城河

- **產品廣度：** 從主機板／顯卡的板卡能力延伸至 AI、HPC、Cloud、5G server、機櫃與冷卻方案，可服務雲端與地端運算；產品廣度已獲一手文件支持，但客戶黏著度與超額報酬仍未被量化。[S1][S3]

- **平台整合：** GB300 NVL72 已列為 production，GIGAPOD 搭配 GPM 提供監控、排程與資源管理；若軟硬整合能提高部署效率，價值可能高於單機出貨，但文件未拆方案收入、毛利或續約性。[S3]

- **結構弱點：** 關鍵處理器／GPU 平台仰賴 Intel、NVIDIA、AMD，且產品世代短、匯率與原料／運輸成本波動大；公司具多產品線可分散終端需求，卻不能消除上游集中與技術換代風險。[S1]

## 成長動能、催化劑與驗證 KPI

- **SERVER 組合提升：** 2026Q1 占比升至 71.5%；下一季應同看 SERVER 占比、合併營收、毛利率與營業利益率。若占比續升但毛利與現金流走弱，規模成長未必等同經濟利益改善。[S2][S3]

- **AI server 需求：** 管理層稱當季主要成長動能為 AI server、年增 110%；驗證 KPI 是後續公司是否持續揭露 AI server 增速、量產平台與可辨識收入，而不是以全部 SERVER 71.5%代替 AI 占比。[S3]

- **機櫃與液冷：** GB300 NVL72、HGX B300 液冷 GPU server 與 GIGAPOD 是近期產品催化；應觀察 production／開發狀態、實際交付、存貨週轉及平台切換，Vera Rubin 在文件中仍只是 early-stage development。[S3]

- **全球產能與交付：** 法說列出台灣南平、中國東莞／寧波、美國洛杉磯及馬來西亞柔佛等製造據點；KPI 是新據點量產、跨區交付與資本支出效益，若產能先開而訂單爬坡延後，固定成本與營運資金會上升。[S3]

## 風險與預警指標

- **AI 占比不可見：** 公司揭露 SERVER 與 AI server 增速，未揭露 AI-only 金額、占比、客戶或毛利；任何把 71.5%直接視為 AI 收入的分析都會越過一手證據。[S3]

- **庫存與平台週期：** 存貨年增 124.88%，高於營收增幅；若 GPU／CPU 平台切換、零組件交期或客戶驗收改變，存貨跌價、訂單遞延與現金占用可能同時上升。[S1][S2][S3]

- **供應與地緣政治：** 主要平台晶片集中於少數國際供應商，年報並列稅率、匯率、原料與運輸成本為不利因素；預警指標是採購集中、應付帳款、交期、出口限制與製造區域調整。[S1][S2]

- **客戶資訊缺口：** 三份核心文件沒有提供伺服器前五大客戶、單一 CSP 占比或訂單取消條款；在 SERVER 占比快速提高時，客戶集中度是必須於後續財報／法說補查的重大缺口。[S1][S2][S3]

## 管理層與治理

2024 年報列董事會、審計委員會與內部稽核／會計師定期溝通機制，並記錄海外可轉債、現增、營業用不動產與子公司投資等議案；這些制度可驗證，但 2026 年資本配置成效仍應以最新財報的現金流、負債及投資回報判斷。[S1][S2]

法說把營收續創高、產品組合優化與 AI 雲端到邊緣覆蓋列為 2026 展望；這是管理層前瞻陳述，不是已實現結果。後續應對照實際 SERVER 占比、毛利率、庫存與營業現金流，避免用單一營收目標替代治理與執行評估。[S3]
