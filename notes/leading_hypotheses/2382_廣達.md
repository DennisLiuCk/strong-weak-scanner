- **市場主張：** Taipei Times 轉述廣達預期 2026Q2 AI server 營收季增雙位數、全年 AI server 營收維持三位數成長，非凡新聞則報導 H1 合併營收已達 1.8458 兆元；研究端據此提出更嚴格但尚無公司正式指引的命題：2026H2 合併營收高於 H1。
- **消息日期：** 2026-05-15；H1 路徑更新至 2026-07-07。
- **研究收錄：** 2026-07-18（前瞻捕捉；未回填為消息日）。
- **來源層級：** 一條是英文媒體轉述公司法說，一條是媒體報導已公告月營收；兩者都不是廣達正式的 H2 合併營收財測。H2 大於 H1 是研究端的可證偽推論，不是管理層原話。
- **目前狀態：** 合理線索・證據不足（`plausible_lead`）。
- **正式資料基準：** 已獨立驗證的正式筆記顯示，廣達 2025 年營收 2.1237 兆元；2026Q1 營收 8,092.21 億元，但毛利率僅 4.78%、營業現金流出 922.10 億元，且期末應收與存貨合計較年初增加 3,249.05 億元。AI server 年增翻倍是年報揭露，公開文件仍未提供 AI-only 絕對營收。
- **可證偽條件：** 依公司公告的 2026 年 7–12 月各月合併營收加總，若未高於 1–6 月的 1,845,814,207 千元，假說即不成立；採驗證日可得的最新更正值，不以單月新高、AI server 成長率、出貨量或股價替代。
- **驗證期限：** 2027-01-15。
- **下次驗證：** 每月只更新 H2 累計路徑；待 2026 年 12 月營收公告後一次定案，並另以 Q2–Q4 毛利率、應收、存貨、借款與營業現金流檢查成長品質。
- **研究判讀：** 支持路徑是 AI server、ASIC 與下一代平台交付延續高成長，使 H2 絕對營收超越已創高的 H1；主要反證是客戶驗收或平台時程遞延、關鍵料件限制、筆電轉弱，或大額工作資金需求限制交付。市場已有三位數 AI server 成長敘事，不能假設期待尚未反映；缺少一致預期與同口徑估值快照，也不能宣稱 H2 超越 H1 會構成誤價，現階段應等待營收與現金轉換共同證明。
