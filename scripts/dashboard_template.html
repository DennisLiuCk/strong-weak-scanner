每個交易日收盤後，把同族群個股放上同一把尺：比較價格與抗跌、法人原始買賣及其同業位置、融資槓桿，再依公開規則分層。每個分數都可點開，看原始數據與判讀；資料來源見頁尾。

今日重點台積電專區族群比較個股分層族群內個股數據解剖指南

觀察層 · 不計分

交易／部位觀察指南 · 官方資料怎麼讀

這組數字回答「今天的成交與部位是怎麼形成」，不回答「明天一定漲或跌」。點開個股或族群明細裡的「數據解剖」，可看到當日實際數字、公式與有效樣本。

先看交易與法人

平均每筆成交
成交量／成交筆數；成交金額／成交筆數把量增拆成「筆數變多」或「平均單筆變大」。成交筆數不是投資人帳戶數，不能據此認定散戶或法人增加。
法人方向強度
(買進−賣出)／(買進＋賣出)，範圍−100%～+100%+100%代表只有買進，−100%代表只有賣出，0%代表買賣相等。同樣淨買超，低方向強度可能只是雙向交易都很大。
法人總活動占比
四類法人買進＋賣出／(2×成交量)市場每筆成交同時有買賣兩側，因此分母要乘2。高占比只說法人參與多，方向仍要回頭看方向強度。
自營自行／避險
自行買賣與避險分開列示自行較接近方向性庫存；避險可能來自權證、ETF或套利。避險淨賣不能直接解讀為自營商看空。

再看部位、限額與市場基準

融資餘額來源
今日餘額−同一份官方日報的前日餘額＝資買−資賣−現償分辨融資下降是賣出部位，還是用現金償還；融券則是券賣−券買−券償。採用日報內的前日餘額，可納入交易所對基期的調整。
官方信用限額
融資／融券餘額÷交易所公告限額描述信用額度使用程度；不同於既有「散戶水位」的融資餘額÷發行股數。
外資法令空間
外資持股÷法令容許股數；另列尚可投資股數接近100%表示法規空間較少，不代表外資必然繼續買進或賣出。
借券賣出拆解
新增賣出−還券＋調整＝餘額變動若變化主要來自「調整」，不應直接當成空方新部位；今天的新增賣出應對照昨天公告的次日限額。
官方指數超額報酬
個股還原報酬−同市場官方含息指數報酬上市股扣TWSE、上櫃股扣TPEx；正值表示跑贏市場。族群先逐檔扣除各自基準，再取中位，避免上市／上櫃混合造成偏差。

建議順序：方向強度回答「買賣哪一側較多」→ 活動占比回答「法人參與多少」→ 流量恆等式回答「餘額為何改變」→ 限額回答「空間還有多少」→ 超額報酬回答「扣除市場後是否仍跑贏」。缺值顯示「-」，代表資料或所需歷史不足，不代表0。

資料來源：TWSE／TPEx 官方每日批次與公告（價格、三大法人、融資券、借券、外資持股、處置／注意），FinMind（月營收、財報、市場事件與台積電參考數據）。所有分數皆為族群內相對排名；觀察層（台積電專區）不計分。計算規則與原始碼見 GitHub repo。
